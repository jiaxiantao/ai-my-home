# Mainstream Car Models

Put licensed mainstream car GLB files in this folder with the names below:

- `suv-mainstream.glb`
- `sedan-mainstream.glb`
- `offroad-mainstream.glb`
- `mpv-mainstream.glb`

The showroom page auto-detects these files and will prioritize them over fallback demo assets.

Current installed models (downloaded and ready):

- `suv-mainstream.glb`: Poly Pizza "SUV" by Quaternius (CC0 1.0)
- `sedan-mainstream.glb`: Poly Pizza "Car" (CC0 1.0)
- `offroad-mainstream.glb`: Poly Pizza "Rover" (CC0 1.0)
- `mpv-mainstream.glb`: Poly Pizza "Stationwagon" (CC0 1.0)
